h2. Pattern for file name.

Some patterns used here. The main goal is make it easy to understand, find and reuse file, etc.

A## := Lista de exercício, trabalhos, tarefas;
_solution := resposta e soluções;
A##_E### := Exercício (exercícios dependem de uma lista)
Q# := provas
_grade := notas
<AULA#>_<SIGLA_DISCIPLINA>_TLN_<ITEM_AULA#> := (Teacher's Lecture Notes) notas de aula - professor
<AULA#>_<SIGLA_DISCIPLINA>_TLNE_<ITEM_AULA#>_<NOME_ROIGINAL> := material de apoio complementar a determinada aula
<SIGLA_DISCIPLINA>_RESEARCH_<NOME_ORIGINAL> := material de pesquisa extra
chap## := capítulo de algum livro, etc;
pg#### := página de algum livro, etc
S## := Seminário
EXP## := experimento
<SIGLA_DISCIPLINA>_syllabus := programa de estudo / currículo
_calendar := calendário
@_backup_@@-##-####.zip := backup de arquivos. Exemplo: math-ufscar-2011-1_backup_DEC-17-2011
